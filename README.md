# Farma Nossa - Backend do Gerenciador de estoque de Farmacias
- Cadastro de usuários
- cadastro de produtos
- Cadastro de locais
- Lançamento de entradas (com lote)
- Lançamento de saídas
- Consulta de estoque
- 4 relatórios (que podem ser exportador para PDF)
